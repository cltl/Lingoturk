Instructions

In this task, conversation threads consisting of a number of comments will be provided to you. Comment 1 indicates the beginning of a new conversation thread. Comments are numbered in the given order such that every comment is a reply to its previous comment if it is not the first comment of the thread. User IDs are also displayed to identify whom each user is replying to. Here is an example:

Comment 1

Comment 2 (reply to Comment 1)

Comment 3 (reply to Comment 2)

..................

Comment n (reply to Comment n-1)

Every comment is displayed on a separate page. The context containing all previous conversations will be displayed to you provided that the current comment is not the first comment in the conversation thread. The user ID of each comment in the conversation thread constituting the context is displayed at the beginning of the corresponding comment as follows:

User ID 1_Comment 1 - User ID 2_Comment 2 - User ID 1_Comment 3 - User ID 2_Comment 4

You should first read the context. Then, read the following comment carefully and identify the words that explicitly make the comment inappropriate, such that the comment expresses an improper, negative, bad, unfavorable, or unpleasant judgement. Explicitly inappropriate words are words that are inappropriate in all cases and one does not need to know the sentence in which they are used in order to be able to understand that they are inappropriate, such as swear words and slurs. The ID of each word is written right at the beginning of it. In case a comment has absolutely no explicitly inappropriate word, please check "No explicitly inappropriate word". Please note that you should not leave this section completely unannotated. Below are some examples of such language:

1- She is not being a bitch. She is just less likely to put up with your shit.

Explicitly inappropriate words: bitch, shit

2- Fuck you!

Explicitly inappropriate words: Fuck

3- You are a monkey nigga.

Explicitly inappropriate words: nigga

After having identified the explicitly inappropriate words, you must decide whether the comment conveys an implicitly inappropriate meaning as well. Implicitly inappropriate words refer to words that are not inappropriate in themselves, but when used in certain situations and sentences, can have or imply an inappropriate meaning. The ID of each word is written right at the beginning of it. In case a comment has absolutely no implicitly inappropriate word, please check "No implicitly inappropriate word". Please note that you should not leave this section completely unannotated. Below are some examples of such language:

1- Let's kill jews and kill them for fun.

Implicitly inappropriate words: kill, kill

2- All of them are cockroaches who live off of welfare benefits and need to be taken away.

Implicitly inappropriate words: cockroaches, need, to, be, taken, away

3- You are a monkey nigga.

Implicitly inappropriate words: monkey

As can be seen, a comment can contain both explicitly and implicitly inappropriate words.
